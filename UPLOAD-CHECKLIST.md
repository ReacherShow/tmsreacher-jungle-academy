# Upload checklist

Upload the contents of this folder to the GitHub repository root.

Include:
- `src/`
- `public/`
- `docs/`
- `.env.example`
- `.gitignore`
- `.nvmrc`
- `firebase.json`
- `firestore.rules`
- `index.html`
- `netlify.toml`
- `package.json`
- `README.md`
- `vite.config.js`

Do not upload:
- `node_modules/`
- `dist/`
- `package-lock.json`
- the downloaded ZIP file
- any `.env` file
- any Firebase service-account or Admin SDK key

Before the first account test, publish `firestore.rules` and bootstrap the admin account using `docs/FIREBASE-BETA-SETUP.md`.

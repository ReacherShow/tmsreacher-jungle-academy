# Learning Design Direction

## Goal

The goal is not merely faster fact recall. Jungle Academy should help a learner understand what an operation means, choose a useful strategy, explain their reasoning, retrieve knowledge later, and apply it in unfamiliar situations.

## Core learning loop

### 1. Retrieval
The learner attempts the problem before seeing the solution.

### 2. Clear feedback
Correct answers receive a large, celebratory confirmation. Incorrect answers show the complete equation and keep the moment on screen long enough to process it.

### 3. Representation and strategy
The learner can choose among visual groups, skip counting, break-apart equations, number lines, fraction models, and place-value models.

### 4. Guided retry
After a miss, the learner solves the exact problem again instead of immediately escaping to a new question.

### 5. Metacognition
After using help, the learner identifies which strategy was useful. This preference can later guide the order of hints.

### 6. Spaced review
Each fact receives a future review date. Correct retrieval moves the next review farther away. A miss brings the fact back sooner.

## Next learning features

- 12-minute guided expeditions with warm-up, teaching mission, adaptive practice, transfer challenge, and reward
- Short word problems and real-world applications
- “Teach Purple” moments where the learner explains a strategy
- Confidence checks before answers
- Mixed-topic review to develop strategy selection
- Mastery based on accuracy across time, not one successful session
- Parent-facing reports that separate recall, understanding, strategy use, and transfer
- Content review by an elementary mathematics educator before public release
